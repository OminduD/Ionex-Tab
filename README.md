### ğŸ”§ Manual Installation (For Developers)

#### Prerequisites
```bash
Node.js >= 16.0.0
npm >= 8.0.0
```

#### Clone Repository
```bash
git clone https://github.com/OminduD/Ionex-Tab.git
cd Ionex-Tab
npm install
```

#### Build for Chrome/Edge/Brave
```bash
npm run build:chrome
```

Then:
1. Open `chrome://extensions/`
2. Enable **Developer mode** (top right toggle)
3. Click **Load unpacked**
4. Select the `dist/` folder
5. Open a new tab! ğŸ‰

#### Build for Firefox
```bash
npm run build:firefox
```

Then:
1. Open `about:debugging#/runtime/this-firefox`
2. Click **Load Temporary Add-on**
3. Select `dist/manifest.json`
4. Open a new tab! ğŸ‰

**Note:** Firefox temporary add-ons are removed on browser restart. For permanent installation, install from [Firefox Add-ons](https://addons.mozilla.org/en-US/firefox/addon/ionex-tab-enhanced-new-tab/).

#### Build Both Versions
```bash
npm run build:all
```
Creates:
- `dist/` - Chrome version
- `dist-firefox/` - Firefox version

#### Development Mode
```bash
npm run dev
```
Opens Vite dev server at `http://localhost:5173` (not a browser extension, for component testing)
